Kaviya Sree N — Portfolio Website

This project is a responsive, multi‑page personal portfolio website created using plain HTML, CSS, and vanilla JavaScript. It is designed to present a Computer Science Engineering undergraduate’s professional profile in a structured, interactive, and visually appealing way — essentially turning a resume into a navigable website.

🔹 Purpose
The site serves as a digital portfolio to showcase:

Academic background and certifications

Technical skills and problem‑solving achievements

Internship and open‑source contributions

Featured projects with interactive visualization

Contact information with a functional message form

🔹 Pages & Content
Home (index.html): Hero intro with name, role, tagline, stats, and navigation cards.

About (about.html): Summary, education timeline, certifications, and quick‑reference details.

Skills (skills.html): Categorized skills grid, coding stats (LeetCode, SkillRack, CodeChef), and achievements.

Experience (experience.html): Internship details and open‑source contributions.

Projects (projects.html): Featured project with an exploded stack diagram (frontend, backend, database layers) plus secondary projects.

Contact (contact.html): Contact banner, clickable links (email, phone, LinkedIn), and a mailto‑based message form.

🔹 Design System
Theme: Dark background with vibrant accent palette (violet, pink, teal, emerald, amber).

Typography: Space Grotesk (headings), Inter (body), JetBrains Mono (labels/data).

Layout: CSS Grid + Flexbox, blueprint‑style background, corner tick marks for technical aesthetic.

Motion: Hover transforms, animated stat gauges, all respecting prefers‑reduced‑motion.

🔹 Tech Stack
HTML5 — semantic markup

CSS3 — custom properties, Grid, Flexbox

JavaScript — vanilla, for navigation toggle and form behavior

Google Fonts — Space Grotesk, Inter, JetBrains Mono

No frameworks, no build tools, no external dependencies
